
# Description

A vue app to check reaction timer (similar to the one in Aim Labs). Good to practice before playing FPS games.




## Prerequisites

- [Node.js](https://nodejs.org/en/download/)

- Vue.js/Vite
## Instructions



- Clone this repository 

- Install all the dependencies using 

```bash
npm install

```

- Run the application using 

```bash
npm run serve

```
